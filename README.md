# C3PO

Csharp Command and Control for Pentesting Operations

## Installation
Utiliser le script pour installer

One liner, full install
```
wget -qO- https://raw.githubusercontent.com/Fropops/C3PO/refs/heads/master/Install/install.sh | bash -- All
```

One liner, Only Commander
```
wget -qO- https://raw.githubusercontent.com/Fropops/C3PO/refs/heads/master/Install/install.sh | bash -s -- Commander
```

One liner, Only TeamServer
```
wget -qO- https://raw.githubusercontent.com/Fropops/C3PO/refs/heads/master/Install/install.sh | bash -s -- TeamServer
```
